## 💾 배운 것
viewport width :  화면 크기를 기준으로 한 너비
viewport height : 화면 크기를 기준으로 한 높이
화면의 길이가 1000px일 때 
100vh는 1000px
1vh는 10px

## 📝 메모
TAM에서 각각 브라우저의 크기가 다르므로 css단위를 em이나 rem에서 vw나 vh로 수정해야할꺼 같다.
